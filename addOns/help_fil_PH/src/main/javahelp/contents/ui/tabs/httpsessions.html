Ang mga Sesyon ng HTTP na tab

This tab shows you the set of identified HTTP sessions for each Site, as detected by the HTTP Sessions extension.

Ang kasalukuyang Site na tinutukoy ng impormasyon ay maaring matuklas sa pamamagitan ng toolbar o ang Mga Site na tab.

Ang toolbar ay maglalaan ng button ("Bagong Sesyon") na magpapahintulot sa iyo na magsimula ng bagong sesyon, pumupuwersa sa lahat ng palabas na request na mga mensahe na hindi masali ang mga tokens ng na-set na sesyon, kaya gugunitain ng server na ito ay isang bagong sesyon. Ito ay magpapahintulot ng paggawa ng isang bagong sesyon, na hindi mawawasak ang huling nagawa.

Bawat entry ng Sesyon na table (bawat sesyon) sa pauna ay mayroong malikhang pangalan, pero ito ay maaring mabago sa pamamagitan ng pagpili ng 'Pangalan' na cell at pag-edit nito.

Bawat entry ng Sesyon na table ay maaring makaliwang pindot, na magpapagana ng Popup na Menu, na may sumusunod na mga pagpipilian:

  • Copy Session Token Value - copies the value of the selected session to the clipboard.
  • Remove Session - deletes the session
  • Find Related Messages - Will cause the Search tab to become active, displaying results based on a search with the string displayed in the "Session Tokens' Values" column for the session which was right clicked.
  • Set as active (available only on non-active sessions) - marks this session as active. If any session was previously set as active, it will be unset as active and, if it doesn't specify any token values, it is deleted.
  • Unset as active (available only on the active session) - marks this session as not being active anymore. If the session doesn't specify any token values, it is deleted.
Regarding the active session, more details can be read on the general features help page of the HTTP Sessions extension

Para sa bawat sesyon iyong maaring makita:
  • Aktibo - Kahit ito ay aktibong sesyon o hindi
  • Pangalan - Ang pangalan ng sesyon
  • Mga Balyu ng mga Token ng Sesyon - ang lista ng mga balyus na kaugnay sa bawat mga token ng sesyon. Ang mga entry ay pinaghihiwalay sa ';' na simbolo.
  • Tugmang mga Mensahe - ang bilang ng mga mensahe ng HTTP na natugma sa pamamagitan ng extensyon sa sesyong ito
