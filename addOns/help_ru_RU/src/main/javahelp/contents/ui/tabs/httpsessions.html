Вкладка HTTP-сеансы

На этой вкладке отображается набор идентифицированных сеансов HTTP для каждого сайта, как обнаружено HTTP Расширение сеансов .

Текущий сайт, к которому относится информация, можно выбрать с помощью на панели инструментов или на вкладке Сайты. .

На панели инструментов есть кнопка («Новая сессия»), которая позволяет чтобы начать новую сессию, заставляя все исходящие сообщения запроса быть без набора токенов сеанса, поэтому сервер считает, что это новая сессия. Это позволяет создать новый сеанс, не разрушая старый.

Каждая из записей в таблице сеансов (каждый сеанс) изначально имеет сгенерированное имя, но его можно изменить, выбрав ячейка «Имя» и ее редактирование.

Каждую из записей в таблице Sessions можно щелкнуть правой кнопкой мыши, который активирует всплывающее меню со следующими параметрами:

  • Copy Session Token Value - copies the value of the selected session to the clipboard.
  • Удалить сессию - удаляет сессию
  • Найти похожие сообщения. Откроется вкладка "Поиск". чтобы стать активным, отображая результаты на основе поиска с отображаемой строкой в столбце «Значения токенов сеанса» для сеанса, который был щелкнул правой кнопкой мыши.
  • Установить как активный (доступно только для неактивных сессий) - отметки этот сеанс как активный. Если какая-либо сессия ранее была установлена как активная, он будет отключен как активный и, если в нем не указан какой-либо токен значения, он удаляется.
  • Снять как активный (доступно только в активном сеансе) - помечает этот сеанс как неактивный. Если сеанс не указывает никаких значений токена, он удаляется.
Что касается активного сеанса, более подробную информацию можно прочитать на справочная страница общих функций HTTP-сеансов расширение

Для каждого сеанса вы можете увидеть:
  • Активный — является ли это активным сеансом или нет.
  • Имя — имя сеанса.
  • Значения маркеров сеанса — список значений, связанных с каждый из токенов сеанса. Записи разделяются символом ';' символ.
  • Messages Matched — количество HTTP-сообщений, которые были соответствует расширению с этой сессией
