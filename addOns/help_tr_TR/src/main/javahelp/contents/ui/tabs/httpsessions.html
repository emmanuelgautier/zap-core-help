HTTP Oturumları sekmesi

This tab shows you the set of identified HTTP sessions for each Site, as detected by the HTTP Sessions extension.

Geçerli Site bilgilerinin değindiği araç çubuğundan veya Siteler sekmesi yolu ile seçilebilir.

Bu araç çubuğu size ("Yeni Oturum") butonunu sağlar ki bu dışarı giden tüm istek iletilerini oturum belirteci olmadan gitmeye zorlar bu yüzden sunucu bunu yeni bir oturum olarak algılar böylece sizin yeni bir oturum açmanıza izin verir. Bu eskisini yok etmeden yeni bir oturum yaratılmasına izin verir.

Oturumlar masasındaki girilerin her biri (her oturum) başlangıçta üretilen bir isme sahiptir fakat bu, 'İsim' birimi seçilerek değiştirilebilir ve düzenlenebilir.

Oturumlar masasındaki girilerin her biri sağ tıklanabilir, böylece şu seçeneklerle birlikte açılır menü aktifleşir:

  • Copy Session Token Value - copies the value of the selected session to the clipboard.
  • Remove Session - deletes the session
  • Find Related Messages - Will cause the Search tab to become active, displaying results based on a search with the string displayed in the "Session Tokens' Values" column for the session which was right clicked.
  • Set as active (available only on non-active sessions) - marks this session as active. If any session was previously set as active, it will be unset as active and, if it doesn't specify any token values, it is deleted.
  • Unset as active (available only on the active session) - marks this session as not being active anymore. If the session doesn't specify any token values, it is deleted.
Regarding the active session, more details can be read on the general features help page of the HTTP Sessions extension

Görebileceğiniz her talep için:
  • Aktif - Bu aktif bir oturum olsun ya da olmasın
  • İsim - Oturum ismi
  • Session Tokens 'Values - her bir oturum belirteci ile ilişkili değerlerin listesi. Girişler ';' sembolüyle ayrılır.
  • Eşleşmiş Mesajlar - bu oturumdaki uzantıya göre eşleşen HTTP mesajlarının sayısı
