Release 1.3.0

Este release trouxe as seguintes mudanças:

Alterações significativas:

Fuzzing/Difusão

Strings em uma resposta agora podem ser difusas para tentar encontrar vulnerabilidades.
Anti CRSF tokens podem ser detectados e regenerados automaticamente durante a difusão.
Esta funcionalidade é baseada no código do projeto OWASP JBroFuzz.

Certificados SSL Dinâmicos

O suporte para conexões SSL foi melhorado e simplificado.
Usuários agora podem criar seu próprio certificado raiz e distribuí-los em seus clientes HTTP.

Modos daemon e API

Iniciando o ZAP com a opção de linha de comando "-daemon" fará com que ele seja executado em segundo plano no modo 'sem cabeça', significando que nenhuma interface de usuário é exibida.
Uma API inicial foi implementada em XML, JSON e HTML.
Se o ZAP estiver sendo executado como um daemon, então a API é ativada automaticamente; caso contrário, a API deve ser habilitada através da tela Opções de API.
A API pode ser navegada abrindo-se http://zap/ em seu navegador quando ele estiver usando o ZAP como proxy.

Integração Beanshell

O BeanShell é uma shell de Java que pode ser usada para executar scripts BeanShell.
Esses scripts são uma forma simplificada de Java que usam muitos elementos de sintaxe Java, mas em um formato mais simples de script.
Todo o código Java é também válido como código BeanShell.
A integração do BeanShell no OWASP ZAP permite que você escreva scripts usando as funções ZAP e data sets.
Isto pode ser um recurso muito poderoso para a análise de aplicações web.

Internacionalização completa

Todas as strings de exibição (display) agora são totalmente internacionalizadas.

Localização

Pequenas mudanças:

Visualização hexa

The Request, Response tabs now provide a 'Hex View' option which will display the request and response bodies in hex format.

Resultados de Busca

The Search tab now displays all instances of a string found rather than just the first instance in each request or response.

Excluir URLs

URLs can be explicitly excluded from the active scanner, spider and from the proxy.

Suporte de cópia

Most of the panels now provide a 'right click' 'Copy' menu option, including the Port Scan and Brute Force panels.

Suporte de desfazer/refazer (undo/redo)

All of the input fields now support Undo/Redo actions using the operating systems default Undo/Redo accelerators:
  • Windows/Linux: Ctrl+Z / Ctrl+Y
  • Mac OS X: Cmd+Z / Cmd+Shift+Z

Suporte de proxy para varredura de porta e opção de timeout

The Port Scanner can now use the outgoing proxy (if configured) and the timeout in milliseconds can also now be set.

Botões de pausa de solicitação e resposta

There are now 2 'Set Break' buttons to allow breaks to be set on all requests and all responses independently.

Botões para expansão das abas de Sites e Informação

There are now 2 buttons which allow you to switch between having the Sites and Information tabs expanded.

Break tab icon changes colour when breakpoint hit

Timeout ajustável

The Option: Connection screen allows you to set the timeout in seconds to make it easier to test slow applications.

Atualizações de biblioteca

Most of the libraries used by ZAP have been updated to the latest versions.

Um novo ícone :)

Problemas conhecidos:

Mac OS X: SSL dinâmico e Google Chrome

Currently Dynamic SSL is not working when using Google Chrome. This is because of an unresolved known issue with Google Chrome and Mac OS X Keychain. When importing OWASP ZAP's Root CA into the keychain and requesting a SSL website, an "Invalid certificate" error message is shown.
